## Abstract
This repository provides a comprehensive computational framework for evaluating the robustness, sensitivity, and diagnostic capabilities of Artificial Neural Network (ANN)-based **Virtual Sensors** deployed within Digital Twin architectures. 

The framework systematically simulates physical sensor degradation (e.g., stochastic Gaussian noise, deterministic bias) and quantifies the subsequent fault propagation through complex non-linear models. By isolating prediction residuals $r(t) = y(t) - \hat{y}(t)$, the engine extracts high-dimensional time-frequency features and generates publication-ready statistical visualizations to support predictive maintenance and fault isolation research.

---

## 🔬 Core Contributions & Methodology

1. **Deterministic Fault Injection Pipeline**: Enables the precise, scalable injection of synthetic faults into primary sensory inputs (e.g., Intake Manifold Pressure `P_IM`, `EGR_Rate`). The perturbation severity is strictly mapped to the physical baseline of the sensors.
2. **High-Dimensional Feature Extraction**: Converts raw residual timeseries into a 12-dimensional feature matrix via a sliding-window mechanism. Extracted features include time-domain statistics (Mean, Variance, Skewness, Kurtosis), frequency-domain energy (FFT), and transient state indicators (Wavelet coefficients, Spike counts) for downstream ML classifiers.
3. **Statistical Boundary Visualization**: Implements rigorous diagnostic plotting geometries. Residual analysis is visually segmented into semantic confidence intervals based on standard deviations ($\pm 1\sigma, \pm 2\sigma$), providing explicit visual boundaries for anomaly detection.

## 📚 Theoretical Context:

Physical SensitivityEmpirical data generated by this engine underscores the non-linear dependency of ANN architectures on specific thermodynamic variables.For instance, neural network estimations demonstrate catastrophic sensitivity to the Intake Manifold Pressure (P_IM) due to its direct linear dominance in classic mass flow equations ($\dot{m}_{air} \propto P_{IM}$). Conversely, parameters acting as secondary volumetric correctors (e.g., EGR_Rate) exhibit high fault tolerance, with disturbances largely mitigated by the network's mathematical smoothing capabilities. This framework quantifies these phenomena, establishing foundational constraints for fault-tolerant Digital Twin deployment.
